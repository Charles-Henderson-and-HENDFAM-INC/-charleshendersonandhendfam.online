# `<TokenSearch />`

### Warning: component under development, may change in coming weeks

The `<TokenSearch />` is a search component with an optional debounce delay.

If you want to handle debounce delay outside of this component, set `delayMs` to `0`.

## Usage

Example of using [`getTokens`](https://onchainkit.xyz/token/get-tokens) with `<TokenSearch />`.

:::code-group

```tsx [code]
import { TokenSearch } from '@coinbase/onchainkit/token';

// example of async onChange handler
const handleChange = useCallback((value) => {
  async function getData(value) {
    const tokens = await getTokens({ search: value });

    // Do something with tokens here
  }

  getData(value)
}, [])

<TokenSearch onChange={handleChange} delayMs={200} />; // [!code focus]
```

Example of using `<TokenSearch />` with a fixed list of tokens

:::code-group

```tsx [code]
import { TokenSearch } from '@coinbase/onchainkit/token';

const tokens = [
  // ...
];

const handleChange = useCallback(
  (value) => {
    const filteredTokens = tokens.filter(({ name, symbol, address }) => {
      const v = value.toLowerCase();
      return (
        name.toLowerCase().includes(v) ||
        symbol.toLowerCase().includes(v) ||
        address.toLowerCase().includes(v)
      );
    });

    // Do something with filteredTokens here
  },
  [tokens],
);

<TokenSearch onChange={handleChange} delayMs={200} />; // [!code focus]
```
