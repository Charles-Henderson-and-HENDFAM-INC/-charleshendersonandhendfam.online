# `<TokenImage />`

The `TokenImage` component is an image that crops token image to a circle with an adjustable size. When `token` props has no image, render partial token symbol and deterministic dark color.

:::code-group

<div data-title="Preview">
  <ComponentPreview>
    <TokenImage
      token={{
        name: 'Ethereum',
        address: '',
        symbol: 'ETH',
        decimals: 18,
        image: 'https://wallet-api-production.s3.amazonaws.com/uploads/tokens/eth_288.png',
        blockchain: 'eth',
        chainId: 8453,
      }}
      size={32}
    />
  </ComponentPreview>
</div>

```tsx [Code]
import { TokenImage } from '@coinbase/onchainkit/token';

const token = {
  name: 'Ethereum',
  address: '',
  symbol: 'ETH',
  decimals: 18,
  image: 'https://wallet-api-production.s3.amazonaws.com/uploads/tokens/eth_288.png',
  blockchain: 'eth',
  chainId: 8453,
};

export function TokenImageDemo() {
  return <TokenImage token={token} size={32} />;
}
```

:::
