# `<TokenAmountInput />`

The `TokenAmountInput` component is a stylized input field designed for users to specify the amount of a particular token they wish to swap. This component integrates the `TokenSelector` component to allow users to select different tokens.

## Usage

:::code-group

```tsx [code]
<TokenAmountInput
  token={{
    address: '0x1234',
    chainId: 1,
    decimals: 18,
    image:
      'https://dynamic-assets.coinbase.com/dbb4b4983bde81309ddab83eb598358eb44375b930b94687ebe38bc22e52c3b2125258ffb8477a5ef22e33d6bd72e32a506c391caa13af64c00e46613c3e5806/asset_icons/4113b082d21cc5fab17fc8f2d19fb996165bcce635e6900f7fc2d57c4ef33ae9.png',
    name: 'Ethereum',
    symbol: 'ETH',
  }}
  label="Sell"
  amount="3.75"
  onAmountChange={() => {}}
  onTokenSelectorClick={() => {}}
/>
```
