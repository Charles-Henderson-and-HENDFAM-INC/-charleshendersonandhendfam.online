# `<TokenChip />`

The `TokenChip` component is a button that displays the token symbol.
When pressed, it uses the token as the argument to the given click handler.

## Usage

:::code-group

```tsx [code]
import { TokenChip } from '@coinbase/onchainkit/token';
<TokenChip token={token} onClick={handleClick} />; // [!code focus]
```
