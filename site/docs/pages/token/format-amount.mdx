# `formatAmount` [Utility designed for consistent number formatting.]

## Usage

```tsx
import { formatAmount } from '@coinbase/onchainkit/token';

const amount = formatAmount(10000, { minimumFractionDigits: 2 });
> '10,000.00'; // if in U.S. English locale // [!code hl]
> '10.000,00'; // if in EU country locale // [!code hl]
```

## Returns

`string` - The formatted amount.
