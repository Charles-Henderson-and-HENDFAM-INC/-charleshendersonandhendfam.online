# `<TokenSelector />`

### Warning: component under development, may change in coming weeks

The `<TokenSelector />` component renders a list of [`<TokenChip />`](https://onchainkit.xyz/token/token-chip) and [`<TokenRow />`](https://onchainkit.xyz/token/token-row) components with a search box.

## Usage

:::code-group

```tsx [code]
import { TokenSelector } from '@coinbase/onchainkit/token';

const tokens = [
  // ...
];

<TokenSelector tokens={tokens} onSelect={} onClose={} />; // [!code focus]
```
