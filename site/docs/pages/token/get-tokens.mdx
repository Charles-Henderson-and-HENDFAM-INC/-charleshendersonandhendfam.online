# `getTokens`

The `getTokens` retrieves a list of tokens on Base by searching for name, symbol or address of a token.

## Usage

:::code-group

```tsx [code]
import { getTokens } from '@coinbase/onchainkit/token';

const tokens = await getTokens({ limit: '1', search: 'degen' });
```
