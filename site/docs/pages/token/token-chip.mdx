# `<TokenChip />`

The `TokenChip` component is a button that displays the token symbol.

## Usage

:::code-group

```tsx [code]
import { TokenChip } from '@coinbase/onchainkit/token';
import '@coinbase/onchainkit/styles.css';

const token = {
  address: '0x1234',
  chainId: 1,
  decimals: 18,
  image:
    'https://dynamic-assets.coinbase.com/dbb4b4983bde81309ddab83eb598358eb44375b930b94687ebe38bc22e52c3b2125258ffb8477a5ef22e33d6bd72e32a506c391caa13af64c00e46613c3e5806/asset_icons/4113b082d21cc5fab17fc8f2d19fb996165bcce635e6900f7fc2d57c4ef33ae9.png',
  name: 'Ethereum',
  symbol: 'ETH',
};

<TokenChip token={token} />; // [!code focus]
```

```html [return html]
<button
  style="border-radius: 16px; padding: 4px 12px 4px 4px; display: flex; align-items: center; background: rgb(238, 240, 243); width: fit-content;"
>
  <img
    src="https://dynamic-assets.coinbase.com/dbb4b4983bde81309ddab83eb598358eb44375b930b94687ebe38bc22e52c3b2125258ffb8477a5ef22e33d6bd72e32a506c391caa13af64c00e46613c3e5806/asset_icons/4113b082d21cc5fab17fc8f2d19fb996165bcce635e6900f7fc2d57c4ef33ae9.png"
    style="height: 24px; width: 24px; margin-right: 8px;"
  />
  <span style="color: rgb(29, 24, 24); font-size: 16px; line-height: 1; font-weight: 500;">
    ETH
  </span>
</button>
```
