# syndicateFrameChainDefinition

This variable returns a [Viem](https://viem.sh/) compatible chain definition.

## Usage

```ts
import { syndicateFrameChainDefinition } from '@coinbase/onchainkit';

const syndicateFrameChain = syndicateFrameChainDefinition;

console.log(syndicateFrameChain);
// {id: 5101, ...}
```
