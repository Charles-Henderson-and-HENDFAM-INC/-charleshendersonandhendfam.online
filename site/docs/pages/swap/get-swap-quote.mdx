# `getSwapQuote`

The `getSwapQuote` function is used to get a quote for a swap between two assets.

## Usage

:::code-group

```tsx [code]
import { getSwapQuote } from '@coinbase/onchainkit/swap';

const quote = await getSwapQuote({ ... });
```

```ts [return value]
...
```

:::
