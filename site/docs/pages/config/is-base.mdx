# `isBase` [Utility that verifies if the chain id is a valid Base or Base Sepolia chain id.]

## Usage

```tsx
import { isBase } from '@coinbase/onchainkit';

const baseChainId = 8453;
isBase({ chainId: baseChainId });
> true; // [!code hl]

const baseSepoliaChainId = 84532;
isBase({ chainId: baseSepoliaChainId });
> true; // [!code hl]

const ethereumMainnetChainId = 1;
isBase({ chainId: ethereumMainnetChainId });
> false; // [!code hl]
```

## Returns

`boolean` - Returns `true` if the chain id is Base or Base Sepolia, otherwise `false`.
