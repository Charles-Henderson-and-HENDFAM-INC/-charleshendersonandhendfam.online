# `<Badge />`

Badging is used to show that a user has an attestation attached to their account.

## Usage

Badge with default colors:

:::code-group

```tsx [code]

<Badge />; // [!code focus]
```

## Badge on `<Name />` and `<Avatar />`

Badge with [`<Name />`](/identity/name), best used when [`<Name />`](/identity/name) are displayed alongside [`<Avatar />`](/identity/avatar) components.

:::code-group

```tsx [code]
import { base } from 'viem/chains';
import { OnchainKitProvider } from '@coinbase/onchainkit';
import { Name } from '@coinbase/onchainkit/identity';

const address = '0x02feeb0AdE57b6adEEdE5A4EEea6Cf8c21BeB6B1';
const schemaId = '0x1801901fabd0e6189356b4fb52bb0ab855276d84f7ec140839fbd1f6801ca065';

<OnchainKitProvider chain={base} schemaId={schemaId}>
  <Name address={address} showAttestation />
</OnchainKitProvider>;
```
