# `<Name />` [Displays the ENS name associated with an Ethereum address]

:::code-group

<div data-title="Preview">
  <ComponentPreview>
    <Name address="0x02feeb0AdE57b6adEEdE5A4EEea6Cf8c21BeB6B1" />
  </ComponentPreview>
</div>

```tsx [Code]
import { Name } from '@coinbase/onchainkit/identity';

export function NameDemo() {
  return <Name address="0x02feeb0AdE57b6adEEdE5A4EEea6Cf8c21BeB6B1" />;
}
```

:::

```tsx
import { Name } from '@coinbase/onchainkit/identity';
```
