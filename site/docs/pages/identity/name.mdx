# `<Name />`

The `Name` component is used to display ENS names associated with Ethereum addresses.

## Usage

Address with an ENS:

:::code-group

```tsx [code]
import { Name } from '@coinbase/onchainkit/identity';
<Name address="0x02feeb0AdE57b6adEEdE5A4EEea6Cf8c21BeB6B1" />; // [!code focus]
```

```html [return html]
<span>zizzamia.eth</span>
```

:::

<App>
  <Name address="0x02feeb0AdE57b6adEEdE5A4EEea6Cf8c21BeB6B1" />
</App>

Address with an ENS avatar, showing his address sliced:

:::code-group

```tsx [code]
import { Name } from '@coinbase/onchainkit/identity';
<Name address="0x02feeb0AdE57b6adEEdE5A4EEea6Cf8c21BeB6B1" showAddress />; // [!code focus]
```

```html [return html]
<span>0x123...5678</span>
```

:::

<App>
  <Name address="0x1234567890abcdef1234567890abcdef12345678" showAddress />
</App>

Address without an ENS avatar, and it defaults to his sliced address:

:::code-group

```tsx [code]
import { Name } from '@coinbase/onchainkit/identity';
<Name address="0x1234567890abcdef1234567890abcdef12345678" />; // [!code focus]
```

```html [return html]
<span>0x123...5678</span>
```

:::

<App>
  <Name address="0x1234567890abcdef1234567890abcdef12345678" />
</App>
