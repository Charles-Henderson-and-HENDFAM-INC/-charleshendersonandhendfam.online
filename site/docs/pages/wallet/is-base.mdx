# `isBase`

The `isBase` utility is designed to verify if the chain id is a valid base or base sepolia chain id.

## Usage

:::code-group

```tsx [code]
import { isBase } from '@coinbase/onchainkit/wallet';

const isBaseResponse = await isBase({ chainId: '8453' });

if isBaseResponse.isValid {
  console.log('Chain id is a valid base or base sepolia chain id');
}
```

```ts [return value]
[
  {
    isValid: true,
  },
];
```

:::
